DevOpsMind Installer Assets
===========================

Recommended (one command):
--------------------------
curl -fsSL https://infraforgelabs.github.io/meta/devopsmind/download/install.sh | bash

Alternative (manual install):
-----------------------------
pipx install devopsmind

Requirements:
-------------
- Python 3.9+
- pipx

Upgrade:
--------
pipx upgrade devopsmind

Uninstall:
----------
pipx uninstall devopsmind

Notes:
------
- DevOpsMind is installed in an isolated pipx environment
- No system Python files are modified
- Works on Linux and macOS
- Windows users should use WSL2
