Core Projects

InfraForge

InfraForge is a production-grade infrastructure automation platform designed to evolve from integration to full independence — supporting Terraform, Ansible, Docker, Helm, Kubernetes, CI/CD pipelines, security tooling, and GitOps workflows.

DevOpsMind

DevOpsMind is an offline-first, hands-on DevSecOps learning engine that teaches Linux, CI/CD, cloud, containers, Kubernetes, and security through real-world challenges — not passive tutorials.
