Sustainability & Funding

All InfraForge Labs projects are free to use. Payment is never required to access functionality.

Optional paid offerings support:

  • Long-term maintenance
  • Documentation & education
  • Security reviews & QA
  • Support guarantees

Paid offerings never unlock features or change execution behavior. All users run the same code.
