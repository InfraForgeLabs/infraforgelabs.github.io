Sustainability & Funding

All InfraForgeLabs projects are free to use. Payment is never required to access functionality.

Optional paid offerings exist only to support:

  • Long-term maintenance and stability
  • Documentation and learning material
  • Security reviews and quality assurance
  • Support and response guarantees

Paid offerings never unlock features or alter execution behavior. All users run the same code.
